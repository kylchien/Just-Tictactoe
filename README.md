# Just-Tictactoe
<body>
Just a Tic-Tac-Toe game with classic AI algorithm practices.<br>

So far it supports:<br>
- MaybePerfect (Newell and Simon's algorithm)
- Minimax
- Negmax (with alpha-beta pruning)


Testing with Qt 5.5 and MinGW 4.5
